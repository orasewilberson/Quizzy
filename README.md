# Quizzy - Application Android de Quiz

## Description
Quizzy est une application Android qui permet de :
- Jouer à des quiz par catégorie
- Suivre son score
- Voir l’historique des scores
- Supprimer ses scores
- Choisir un thème clair ou sombre

## Fonctionnalités
- **Sélection de catégorie** : Programmation, Science, Culture générale, Mathématiques, Autre
- **Affichage des scores** : pour l’utilisateur ou pour tous les joueurs
- **Gestion du thème** : Clair / Sombre
- **Suppression des scores** pour l’utilisateur connecté
- Interface fluide avec boutons de réponse animatifs
